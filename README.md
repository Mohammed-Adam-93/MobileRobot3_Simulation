# MobileRobot3_Simulation
![MobileRobot3_Simulation_image_0](images/0_the_Bot.png)

# Goal
The goal of this project is to simulate the robot Turtle3 that represent a ready to use setup to implement and test advance control and navigation programs and algorithms, the robot should:<br>
- Simulate TurtleBot3 and all its physical components.<br>
- Enable differntial drive of rear wheels<br>
- Stream live sensor data of an Imu and a Camera<br>
- Spawn the robot in a simulation environment in GAZEBO<br>
